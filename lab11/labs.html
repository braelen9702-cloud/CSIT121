Lab 1 About Me Page In this lab, i learned how to create my first HTML file and publish it online. I previewed the file on my computer, validating it for errors, and uploaded it to GitHub to make it available on the web. On the file itself i sshared information about myself like education, hobbies and goals.
Lab 2 Web Standards Page In lab two i practiced the basics of HTML by hand-coding a simple webpage. I learned how to use standard HTML tags to structure a page and organize the content. this lab helped me understand web standards and how to create a clean readable document.
Lab 3 CSS Page In lab three i learned how to use basic CSS to improve the look of an HTML page. I added simple styling to the pages from lab two which helped me understand how CSS changes a page's design and makes it more appealing.
Lab 4 Image Page In lab four I learned how to add images and use CSS to make a website look more visual and interesting. I used different types of images and styles to improve the design of my page making it look more professional.
Lab 5 My Web In lab five i learned how to make a webpage responsive so it works on different screen sizes. Using lab three, i created more layouts for both mobile and desktop using flexbox, grid or media queries.I also created lab links to help navigate the pages more freely.
Lab 6 Ui and Ux In lab six i learned how to evaluate a website's usability using UI and UX. I reviewed a public website and analyzed its layout, design and it would be experienced for users. This lab helped me understand what makes a website user freindly and how good design improves the experience.
Lab 7 Colors In lab seven I learned how to create and style HTML tables and forms. I built a two page website where one page used a table to maintain information and the other to collect user information. This lab helped me understand how tables display data and how forms can manage information.
Lab 9 Document Tree In lab nine i learned how to code an XML file and build and HTML page that can display the data. I followed an XML tutorial to understand how XML is structured and how i can bring more detail into it.
Lab 10 Did You Know In my final lab I learned how to create a multipage website that explains a web standard of my choice which was DID (Decentralized Identifiers). This lab helped me apply the things I learned using structure tags, designing my web more deatailed and visual appealing While explaining the topic of my choice.
